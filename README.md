NHK News.app
============

auto play NHK News

- using [node-webkit](https://github.com/rogerwang/node-webkit)
- for Mac OSX


## Download

download [zip archive](https://github.com/shokai/nhk-news-app/releases)


## Build

### Requirements

- Ruby
- Node.js

    % gem install bundler
    % bundle install
    % rake npm:install
    % rake

=> generate nhk-news.app
